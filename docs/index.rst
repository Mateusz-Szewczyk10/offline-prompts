    <h2>Why OfflinePrompts?</h2>

.. grid-item::
    :class: top-page-list

    * The first benchmark for OPE/L for prompt-guided sentence personalization

    * End-to-end implementation of data collection, OPE/L, and LLM-based generation

    * Variety of OPE estimators and OPL methods available

    * Movie description personalization using MovieLens

    <h2>OfflinePrompts enables the following tasks</h2>

    <div class="white-space-5px"></div>

    <div class="top-page-list">
        <ul>
            <li>Generating personalized description of movies using LLMs (benchmarks provided)</li>
        </ul>
    </div>

.. card:: 
    :img-top: _static/images/motivate_example.png
    :text-align: center
    
    Example of generating descriptions of "Wall-E (2008)" focusing on "sci-fi" and "romance" aspects.


.. raw:: html

    <div class="white-space-20px"></div>

    <div class="top-page-list">
        <ul>
            <li>Using logged bandit feedback for optimizing the prompt policy for personalization</li>
        </ul>
    </div>

.. card:: 
    :img-top: _static/images/personalized_sentence_generation.png
    :text-align: center
    
    Logged data contains user's context information, (logging) policy's select of prompts, sentence generated by a frozen LLM given the prompt, and reward observed for the generated sentence collected through past system operations.


.. raw:: html

    <div class="white-space-20px"></div>

    <div class="top-page-list">
        <ul>
            <li>Implementing the end-to-end pipeline of prompt-guided language personalization with modularized codes.</li>
        </ul>
    </div>

.. card:: 
    :img-top: _static/images/workflow.png
    :text-align: center
    
    End-to-end implementation workflow of OfflinePrompts, including data collection, policy learning, and deployment.


.. raw:: html

    <div class="white-space-20px"></div>

    <div class="top-page-list">
        <ul>
            <li>Using the implementation of Frozen LLMs provided in HuggingFace smoothly.</li>
        </ul>
    </div>

.. card:: 
    :img-top: _static/images/dataset_module.png
    :text-align: center
    
    Our dataset module standardizes the synthetic/full-LLM simulation procedures, latter of which is compatible with the LLM implementations of HuggingFace.
