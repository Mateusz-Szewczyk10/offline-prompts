Imported as: src.dataset.TransformerRewardSimulator

Parameters:

  • n_users (int) – Number of users.

  • n_items (int) – Number of items.

  • dim_emb (int, default=10) – Dimension of feature embeddings.

  • base_model (PreTrainedModel, default=None) – Base transformer model. When None is given, DistillBERT will be used.

  • tokenizer (PreTrainedTokenizer or PreTrainedTokenizerFast, default=None) – Tokenizer. When None is given, DistillBERTTokenizerFast will be used.

  • device (str, default="cuda") – Device.

  • random_state (int, default=None) – Random state.
