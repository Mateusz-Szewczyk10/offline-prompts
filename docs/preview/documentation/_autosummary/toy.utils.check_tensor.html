  • name (str) – Name of the input tensor.

  • expected_dim (int, default=1) – Expected dimension of the input tensor.

  • expected_dtype ({type, tuple of type}, default=None) – Expected dtype of the input tensor.

  • max_val (float, default=None) – Maximum value allowed in the input tensor.
