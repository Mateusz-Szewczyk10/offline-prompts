Installation
==========

``OfflinePrompts`` is available at `PyPI <https://pypi.org/project/offline-prompts/>`_, and can be installed from ``pip`` or source as follows.

.. card::

    .. tabs::

        .. code-tab:: bash From :class:`pip`

            pip install offline-prompts

        .. code-tab:: bash From source

            git clone https://github.com/aiueola/offline-prompts
            cd offline-prompts
            python setup.py install
